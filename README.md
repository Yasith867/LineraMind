# LineraMind

LineraMind is a full-stack web application that combines AI capabilities with blockchain-inspired data verification.

## Features
- AI-powered question answering (gpt-5.1).
- Simulated Linera microchain metadata for response verification.
- Interactive chat interface.
- AI image generation (gpt-image-1).

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion.
- **Backend**: Node.js, Express, Drizzle ORM.
- **Database**: PostgreSQL.
- **AI**: OpenAI (via Replit AI Integrations).

## Getting Started

### Local Development
1. Clone the repository.
2. Navigate to the project directory: `cd LineraMind`.
3. Install dependencies: `npm install`.
4. Set up environment variables (see `.env.example`).
5. Run the development server: `npm run dev`.

### Deployment
For instructions on deploying the frontend to Netlify, see [DEPLOYMENT.md](./DEPLOYMENT.md).
